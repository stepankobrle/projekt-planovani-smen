# Frontend Instructions

## Stack

- Next.js (App Router)
- TypeScript

## Best Practices

- Use App Router and server components by default – use client components only when necessary.
- Fetch data server-side where possible (Server Components, Route Handlers).
- Use Next.js built-in features: metadata API, image optimization, font optimization.
- Keep components small and focused.
- Use Tailwind CSS for styling (if applicable – confirm before assuming).

## Project Context

- UI pro zaměstnance: zadávání dostupnosti a preference pracovních časů.
- UI pro administrátora: přehled směn, ruční editace, schvalování, export do PDF/.ics.
- Role: ADMIN, EMPLOYEE.
